This file contains the task for custom nginx image with docker compose and push the image to docker hub

1. Pull nginx image
2. create index.html, dockerfile and docker-compose.yml
3. Build and run the image
4. custom nginx images listed
5. custom nginx image running on an container
6. verify the index.html content displayed on web browser under 8080 port
7. login with docker hub credentials and push the custom nginx image
8. verify the custom nginx image uploaded in docker hub web page
